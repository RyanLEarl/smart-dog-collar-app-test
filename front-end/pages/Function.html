• Arduino board reports gyroscope data over wifi network, using MQTT, to AWS IOT Core
  • An IoT Rule is established to listen to the output of the MQTT topic and allow the message to be routed elsewhere
  • Lambda function is triggered when the rule receives new messages, takes json event message, splits it up into the json keys, and generates a message before forwarding it to an sns topic
  • SNS topic can be subscribed to and used to send messages, notifications, and other data to application
